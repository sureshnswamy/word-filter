
# 4letters

it's a real sweet app
that checks the given letters and fetches words containing those lettres. 
## how

### install

```
git clone https://github.com/sureshnswamy/4letters
cd 4letters
npm install
```

## develop

```
node index.js
```

browse to <http://localhost:9966/>.

## test

```
npm test
```

## deploy

```
npm run deploy
```
